There are objects for loading an application which is written in the c4j language. There is a complete datastructure needed to store the application. The objects provides only small integrity checks. It is up to the higher level of the processing to check the data consistency.

All of the objects are prepared to hold the so called declaration reference. This kind of information should be in the human readable form and should be sufficient to identify such objected whenever necessary.

The object XMLHandler is dedicated to load the data structure from the XML document.
