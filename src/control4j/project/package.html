A project is here to encapsulate application files. To create a program for a control4j you can use various formats. And not only that, you can combine all of this formats within one project. And you can do that thanks to the project file.

This package contains classes that provides reading of the project and provides data for futher processing.

In fact project file is a xml file, where the settings of the runtime environment are presented together with list of application files.
